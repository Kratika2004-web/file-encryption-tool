# File Encryption & Decryption Tool

This is a beginner cybersecurity project built using Python.
##  Features
- Encrypt files securely
- Decrypt files using a secret key
- Uses AES-based encryption

##  Tech Used
- Python
- Cryptography library

##  How to Run
1. Install cryptography:
   pip install cryptography

2. Run encryption:
   python encrypt.py

3. Run decryption:
   python decrypt.py
##  Purpose
To understand basic cryptography and data security.
